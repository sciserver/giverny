turbulence - stable release version 1.1.1 (11 January 2022)

version 1.1.1 updates:
    - converted output_array to xarray.
    - simplied retrieving values for a datapoint (or array of datapoints) by using xarray's built-in functionality.
    - improved code for plotting a contour plot to utilize xarray formatting and added contour plot function to backend code.
    - cleaned up backend code memory clearing statements, and moved isotropic cube class instantiation out of the backend code.
    - updated dask backend to be fully automated.
    - refactored code.
